# LFE Grill Me

## Process
1. **Interview**: Ask one question at a time.
2. **Resolve**: Resolve design dependencies one-by-one.
3. **Recommend**: Always provide your recommended answer for each question.
4. **Research**: If a question can be answered by exploring the code/docs, do so instead of asking.
