# Contributing to Library-First Engineering (LFE)

This is an open-source movement to bring architectural rigor to AI-driven development. We welcome contributions to the Manifesto, the repository structure, and the sanitized skill suite.

## How to Contribute
1. **Propose Philosophy Changes**: Open an issue to discuss updates to the Manifesto.
2. **Improve Templates**: Submit PRs for better `.template.md` files or skill definitions.
3. **Share Case Studies**: (Optional) Link to public LFE-compliant repositories.
