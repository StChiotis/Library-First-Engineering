# Architectural Decision Records (ADRs)

> ADRs capture decisions that are hard to reverse, surprising without context, and the result of a real trade-off.
> They live here as numbered sections in this single canonical file — there is no separate ADR directory. New ADRs are appended by the Architect during `/lfe-grill-with-docs` sessions, following the format in `lfe-grill-with-docs/ADR-FORMAT.md`.
